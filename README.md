# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/lukevinicius/hermes
```

```bash
# Install Dependencies
$ bun i
# Run Aplication
$ bun run dev
```
Go to http://localhost:3333 to see the result.
